### 目录说明
* wx: 微信小程序原生demo
* alipay: 支付宝小程序原生demo
* uniapp: uniapp框架demo
* taro: taro框架demo
* mpvue: mpvue框架demo
* wepy2: wepy2.0版本框架demo
* chameleon: chameleon框架demo
* wx-game: 微信小程序原生demo

> 目前只有2.2.0及以上版本的SDK才支持以上第三方框架

目前SDK的线上版本为2.1.0，新版本(2.2.0)正在内测，如需安装新版本，请通过以下命令安装：
```
// 微信
npm i umtrack-wx@next

// 支付宝
npm i umtrack-alipay@next
```
